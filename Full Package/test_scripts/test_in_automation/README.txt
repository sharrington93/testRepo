HOW TO RUN:

	- Place all .csv and/or .txt files with the following params with it's associated value into this folder:

		dist_to_speed_lookup
		dist_to_alt_lookup
		step
		total_time
		wheel_radius
		gearing
		rider_mass
		bike_mass
		gravity
		air_resistance
		air_density
		frontal_area
		rolling_resistance
		top_torque
		top_rpm
		efficiency
		max_distance_travel

	  See template.csv for more details about required file format
	
	- Place the lookup files specified under the params "dist_to_speed_lookup" and "dist_to_alt_lookup" 
	  into the folder "Lookup Files"

	- Open OPTIONS.csv and change the input file and output file names as needed

	- Make sure Automation.py and Simulation.py are in the same directory.  Run Automation.py.
